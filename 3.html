JavaScript

자바 스크립트는 넷스케이프 (그리고 나중에 모질라)에의해 처음 개발된 바와 같이 오랫동안 많은 웹 사이트에대한 웹에 존재했다, 그것은 자바 스크립트가 점점 응용 프로그램에 대한 사용자 인터페이스의 개발에 사용되는 컴퓨팅의 웹 2.0 시대의 상승과 함께 특정 피치를 얻었다, 웹 기반과 데스크톱 기반 모두. 자바스크립트는 CSS와 결합되어 동적 웹 페이지를만들었으며, 이는 Flash기반 웹 사이트에 대한 보다 효율적이고 접근 가능한 대안으로 인기를 끌고 있습니다.
